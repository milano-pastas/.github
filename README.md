# Milano Pastas

Custom software platform designed for pasta manufacturing businesses.

This project digitizes and centralizes key factory operations through an integrated web platform, combining public-facing business information with internal management tools for employees and administrators.

---

## What We Built

### Public Website
Customer-facing website for presenting the business, products, and brand.

### Employee Dashboard
Internal dashboard for factory employees and administrators to manage operations.

### Backend Infrastructure
Backend powered by Supabase for:

- Database management  
- Authentication  
- Data persistence  
- Internal business logic support  

---

## Core Objectives

- Modernize pasta factory workflows  
- Centralize internal operations into one platform  
- Improve employee efficiency through digital dashboards  
- Create scalable infrastructure for future growth  

---

## Tech Stack

- Frontend: React / Vite  
- Backend: Supabase  
- Database: PostgreSQL  
- Authentication: Supabase Auth
